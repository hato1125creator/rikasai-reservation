📚 システム操作マニュアル

全ての機能の使い方を初心者向けに説明します。

STEP 1 — 管理者ログイン

  1. ブラウザで /admin/ を開く
  2. 「ユーザー名」に admin を入力
  3. 「パスワード」に設定したパスワードを入力（初期: rikasai123）
  4. 「ログイン」ボタンを押す

STEP 2 — 生徒が招待リンクを作る流れ

生徒は以下の手順で招待リンクを作成します。管理者の作業は不要です。

  1. /student/ を開く
  2. 「新規登録」タブ → 学校メール（@biblos.ac.jp）と氏名を入力し「認証コードを送信」
  3. メールに届いた6桁のコードを入力してログイン
  4. ダッシュボードで「ゲスト名を入力」→「招待リンクを作成」
  5. 生成されたリンク（例: http://xxx/guest/?token=...）をゲストに送信

STEP 3 — ゲストのQRコード表示

  1. ゲストが招待リンクをタップするだけでQRコードが表示される
  2. 個人情報の入力は不要
  3. QRコードは届け出た際にスタッフがスキャン

STEP 4 — QRコードで入場チェックイン

  1. 「📷 QRスキャン」タブを開く
  2. 「Webカメラ」または「手動入力」を選択
  3. ゲストのQRコードをカメラにかざす or URLを貼り付けて「検証」
  4. 緑の「入場完了しました」の表示と音が鳴ったら成功

⚠ 注意事項

  • メールは @biblos.ac.jp のアドレスのみ有効
  • OTPは10分間有効・5分ごとに3回まで送信可
  • ゲストQRコードは1回しか使えない（再入場不可）
  • 生徒を削除すると、その生徒の招待リンクは全て無効になります
  • 管理者パスワードは8文字以上で設定してください
